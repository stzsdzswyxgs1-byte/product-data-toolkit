【 第一次用 】(只做一次)

▶ 步驟 1: 裝 Python 3.10+
   下載: https://www.python.org/downloads/
   ★ 安裝時必須勾選 [Add Python to PATH]

▶ 步驟 2: 進「商品處理中樞」資料夾
   雙擊 [start.bat]
   會自動:
     - 偵測您的顯卡 (NVIDIA / 沒有)
     - 裝對應的 PyTorch (GPU 版 / CPU 版, 走中國鏡像)
     - 裝 LaMa 去水印模型 (權重已內建, 免下 GitHub)
   等 5-10 分鐘 (主要是 PyTorch ~2GB)
   完成後直接進主介面

▶ 步驟 3: 找管理員開通您的 TG 額度
   1. Telegram 搜尋 @example_admin_bot
   2. 發 /start 拿到「您的 TG ID」
   3. 把 TG ID 給管理員 → 等他批准
