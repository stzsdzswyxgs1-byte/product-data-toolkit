★★★ 整個工具只有「一個按鈕」 ★★★

  📁 商品處理中樞/
      └ 雙擊 [start.bat]   ← 就這個,什麼時候都點它

  第一次點: 自動下載安裝環境 (5-15 分鐘)
  之後再點:  直接開軟件

═══════════════════════════════════════════════

【 第一次用 】(只做一次)

▶ 步驟 1: 安裝 Python 3.10+
   下載: https://www.python.org/downloads/
   ★ 安裝時必須勾選 [Add Python to PATH]

▶ 步驟 2: 雙擊 [start.bat]
   軟件會自動偵測缺什麼套件並自動安裝:
     - 偵測您的顯卡 (NVIDIA / 沒有)
     - 裝對應的 PyTorch (GPU 版 / CPU 版)
     - 裝 LaMa 去水印模型
   等 5-15 分鐘 (看網速)
   完成後會自動進入軟件

▶ 步驟 3: 找管理員開通您的 TG 額度
   1. 打開 Telegram, 搜尋 @example_admin_bot
   2. 發 /start 給機器人
   3. 機器人會告訴您「您的 TG ID」
   4. 把這個 TG ID 給管理員, 等他批准開通

═══════════════════════════════════════════════

【 之後使用 】

▶ 雙擊 [start.bat] 啟動軟件 (不再有等待)
▶ 在主介面填您的 TG ID
▶ 勾選要跑的處理步驟
▶ 選 Excel → 點 [▶ 開始處理]

═══════════════════════════════════════════════

【 配額不夠時 】

去 Telegram @example_admin_bot 發:
   /apply 數量 理由
例:
   /apply 500 跑緊急批次

管理員批准後會自動通知您.

═══════════════════════════════════════════════

【 常見問題 】

Q1. 雙擊 start.bat 沒反應?
A1. 是不是還沒裝 Python? 從 python.org 下載 3.10+,
    安裝時勾選 [Add Python to PATH], 重新雙擊 start.bat.

Q2. 第一次跑 setup 階段失敗?
A2. 多半是網路問題, 重新雙擊 start.bat 即可
    (會跳過已裝部分繼續裝).

Q3. 沒有 NVIDIA 顯卡能跑嗎?
A3. 可以, start.bat 會自動裝 CPU 版, 慢但能跑.
    去水印 LaMa CPU 比 GPU 慢 10-20 倍, 一張圖約 30 秒.

Q4. 顯卡很舊 (2GB / 4GB)?
A4. 軟件會自動偵測 + 調整並發數, 不用管.
    若還是當機, 開軟件後 [⚙ 設定 → LaMa 去水印] 改成 cpu.

Q5. 軟件跑到一半說「配額不足」?
A5. 用 Telegram /apply 申請, 管理員批准後重跑.

Q6. 軟件閃退 / 報錯?
A6. 截圖錯誤訊息給管理員, 帶上 [詳細日誌_*.txt]
    (在輸出目錄).
